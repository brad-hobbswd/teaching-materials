# Little Explorers Learning Hub

Welcome to **Little Explorers Learning Hub**, a free educational resource website created to support Early Head Start, Head Start, Preschool, and Pre-K teachers with affordable, hands-on learning materials.

🌐 **Website:** https://brad-hobbswd.github.io/teaching-materials/

---

## About

Little Explorers Learning Hub provides developmentally appropriate teaching resources designed for classrooms serving children from birth through five years of age.

The goal is to help educators create engaging learning experiences using inexpensive and everyday classroom materials, making quality education accessible for every child and every classroom.
